📊 Посібник з додавання даних

Детальна інструкція для додавання геополітичних подій до платформи

📌 Огляд

Платформа "Геополітичні Події" дозволяє користувачам легко додавати нові геополітичні довідки до інтерактивної карти. Цей посібник розповідає про всі доступні способи та найкращі практики для додавання даних.

💡 Підказка: Ви можете додавати дані трьома способами: через веб-інтерфейс, JSON імпорт або CSV файл.

Що можна додавати?

  • Геополітичні події: Война, конфлікти, політичні зміни
  • Економічні зміни: Торговельні угоди, санкції, кризи
  • Технологічні розробки: Запуски супутників, технологічні інновації
  • Дипломатичні eventi: Саміти, договори, угоди
  • Природні катаклізми: Землетруси, повені, катастрофи

🚀 Розпочнемо

Спосіб 1: Додавання через веб-інтерфейс

1

Відкрийте вкладку "Data Import"

Ліва бічна панель містить вкладку "Data Import" нижче розділу фільтрів.

2

Виберіть файл для імпорту

Натисніть на поле "Пакетний імпорт" і виберіть CSV або JSON файл.

3

Налаштуйте параметри

Виберіть формат дати, режим імпорту (append/replace) та інші параметри.

4

Перегляньте попередній перегляд

Система покаже попередній перегляд даних перед імпортом.

5

Натисніть "Import"

Дані будуть додані до платформи. Ви побачите повідомлення про успіх.

Спосіб 2: Додавання через API (JavaScript)

Якщо ви розробник, ви можете програмно додавати дані:

const newEvent = {
    id: 'unique_id_12345',
    title: 'Назва геополітичної події',
    date: '2024-10-17',
    category: 'Військові конфлікти',
    region: 'Азія',
    country: 'Україна',
    lat: 50.4501,
    lng: 30.5234,
    description: 'Детальний опис події',
    participants: ['Учасник 1', 'Учасник 2'],
    impact: 'Вплив на глобальну економіку',
    importance: 8,
    sources: ['BBC', 'Reuters']
};

app.events.push(newEvent);
app.applyFilters();

📥 Способи імпорту

Режим імпорту

Існує два основні режими для додавання даних:

1. Append (Додавання)

Нові дані будуть додані до існуючих даних.

✅ Коли використовувати: Коли ви хочете розширити набір даних новими событіями.

2. Replace (Заміна)

Существующие дані будуть замінені новими.

❌ Коли використовувати: Тільки коли ви впевнені, що хочете видалити старі дані.

Дедуплікація

Система автоматично знаходить та обробляє дублікати:

  • Auto: Система спробує виявити дублікати за назвою та датою
  • Strict: Суворий режим - повна ідентичність даних
  • Lenient: М'який режим - дозволяє невеликі розбіжності

Формат дати

Система підтримує кілька форматів дати:

  • YYYY-MM-DD (рекомендовано)
  • DD/MM/YYYY
  • MM/DD/YYYY
  • ISO 8601 (YYYY-MM-DDTHH:mm:ssZ)

✅ Перевірка якості даних

Контрольний список для перевірки

Перед додаванням даних переконайтеся, що вони відповідають цим критеріям:

✓

Унікальність ID

Кожне событие має унікальний ID, який не існує в базі даних.

✓

Валідність координат

Широта: -90 до 90, Довгота: -180 до 180

✓

Формат дати

Дата повинна бути у форматі YYYY-MM-DD або іншому підтримуваному форматі.

✓

Коректна категорія

Виберіть одну з визначених категорій:

  • Військові конфлікти
  • Політичні зміни
  • Економічні зміни
  • Технологічні розробки
  • Союзи та договори
  • Тероризм
  • Глобальні кризи

✓

Наявність джерел

Вкажіть щонайменше одне надійне джерело інформації.

✓

Опис та деталі

Запропонуйте детальний та точний опис события.

Звіт про якість

Після імпорту система генерує звіт про якість даних. Ви можете переглянути його у файлі data/quality_report.json.

Метрики якості:

  • Completeness: Відсоток заповнених полів
  • Accuracy: Точність геолокації та дат
  • Consistency: Відповідність категорій та регіонів
  • Validity: Валідність форматів та значень

❓ Поширені питання

Як я можу скасувати добавлення даних?

Ви можете використовувати режим "Replace" для перезавантаження оригінальних даних, або очистити базу дані та завантажити свіжі дані.

Чи можу я редагувати события після додавання?

Так, ви можете редагувати события через веб-інтерфейс, натиснувши на маркер события на карті та натиснувши кнопку "Edit".

Які координати я повинен використовувати?

Використовуйте координати географічного центру события або найважливішої точки. Ви можете використовувати Google Maps або OpenStreetMap для пошуку координат.

Чи можу я імпортувати дані з іншого джерела?

Так, якщо дані мають формат CSV або JSON з потрібними полями, ви можете їх імпортувати. Система проведе валідацію та показатиме попередження про помилки.

Як я дізнаюся про помилки під час імпорту?

Система покаже детальний звіт про помилки після спроби імпорту. Ви можете виправити помилки в файлі та спробувати знову.

Чи є обмеження на кількість событій які я можу додати?

Не, ви можете додавати необмежену кількість событій. Однак для оптимальної продуктивності рекомендується додавати не більше 1000 событій за раз.

Як я можу експортувати дані?

Ви можете експортувати дані у CSV формат через кнопку "Експорт" на панелі управління або використовуючи JavaScript API.

Чи підтримуються мультимовні дані?

Так, система підтримує Unicode та мультимовні дані. Ви можете додавати события українською, англійською, російською або іншими мовами.
